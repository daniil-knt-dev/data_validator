# Валидатор данных (data_validator)

Скрипт для валидации пользовательских данных с гибкой консольной настройкой фильтров.

## 🎯 Что делает скрипт
1. **Настройка:** Принимает от пользователя параметры фильтрации (минимальная/максимальная длина для каждого поля, а также списки обязательных символов).
2. **Проверка:** Фильтрует входные данные на основе созданных правил и проверяет типы данных.
3. **Отчет:** Возвращает результат валидации в виде подробного словаря:
   ```python
   {"username": True, "email": False, "password": True}
   ```

## 🛠️ Текущий функционал и развитие
* На данный момент реализована проверка полей: `username`, `email`, `password`.
* **В планах:** Добавление возможности создания кастомных параметров для проверки (например: `age`, `time`, `city`).
