# Валидатор данных (data_validator)

Скрипт для валидации пользовательских данных с гибкой консольной настройкой фильтров.

## 🎯 Что делает скрипт
1. **Настройка:** Принимает от пользователя параметры фильтрации (минимальная/максимальная длина для каждого поля, а также списки обязательных символов, минимальное/максимальное значение для чисел).
2. **Проверка:** Фильтрует входные данные на основе созданных правил и проверяет типы данных.
3. **Отчет:** Возвращает результат валидации в виде подробного словаря с причиной непрохождения валидации:
   ```python
   {"username": True, "email": (False, "причина"), "password": True}
   ```
---
*Скрипт написан на этапе изучения функций.*
